# ExpenseTracker

A Java-based Personal Expense Tracker console application that allows users to record daily expenses, track monthly reports, and manage financial data efficiently.

## Description

This project is a CLI tool designed to help users track their spending habits. It securely processes:
* Expense Amounts
* Spending Categories
* Dates of Transactions

## Features

* Add and display detailed expense entries
* Generate comprehensive monthly spending reports
* Analyze and identify the highest expense category
* Save and load records using Java Object Serialization

## Technologies Used

* Java
* OOP Concepts
* String Handling
* Loops and Conditions
